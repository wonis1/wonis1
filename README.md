- 문제: 로그인 상태, 학교 등록 여부, 선택 학교에 따라 화면 분기와 접근 경로가 달라지는 구조

- 결과: 인증/등록 상태 기준의 화면 분기 구조를 정리했고, 검색 입력마다 발생하는 불필요한 조회를 줄이는 흐름으로 정리
- 기술: React, TypeScript, TanStack Query, Zustand, React Router, Vite, CSS Module
